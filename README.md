# mytasklist
Simple task list (MEAN App From Scratch - MongoDB, Express, Angular 2 &amp; NodeJS)

### Version
1.0.0

### Installation

```sh
$ npm install
```

### Usage

Start mongodb on local machine:
```sh
$ mongod
```

In other console start backend:
```sh
$ cd mytasklist
$ npm start
```
And one console for starting frontend:
```sh
$ cd mytasklist/client/
$ npm start
```
