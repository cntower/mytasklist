# mytasklist
Task list for registered users. (MEAN, REST with JWT authentifikation)

### Version
1.2.0

### Installation

```sh
$ npm install
```

### Usage

Start mongodb on local machine:
```sh
$ mongod
```

Start backend:
```sh
$ cd mytasklist
$ npm start
```
Start frontend:
```sh
$ cd mytasklist/client/
$ npm start
```
